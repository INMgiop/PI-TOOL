## Qué es Pi

Pi es una capa cognitiva que corre sobre tu código actual.
No reemplaza tu lenguaje ni tu stack. Se instala encima.

Resuelve tres dolores reales:
- **Intención enterrada** — por qué existe una función, no solo qué hace
- **Mapa mental insostenible** — qué se rompe si tocas algo
- **Contexto perdido** — recuperar el hilo después de días sin tocar el proyecto
